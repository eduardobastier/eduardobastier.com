    Projetos

    Curiosidades de Tecnologia

    Este projeto é um site que conta a história do mascote do Android, desenvolvido com HTML e CSS. Ele apresenta uma lista com marcadores personalizados, links externos identificados com emojis, e incorpora iframes para vídeos do YouTube. O design segue uma paleta de cores degradê e usa tags semânticas para melhorar a acessibilidade e SEO. O site é totalmente responsivo, adaptando-se a diferentes tamanhos de tela.

    Cordel Moderno

    Este projeto foi desenvolvido como um desafio do Curso em Vídeo, utilizando HTML e CSS. O efeito parallax foi aplicado às imagens de fundo e a fonte adaptou-se dinamicamente ao tamanho da tela. O resultado foi um site transformado em um 'belo cordel', um estilo literário brasileiro.

    Tela de Login

    Este projeto envolve a criação de uma tela de login com HTML e CSS, que inclui um formulário para inserção de email e senha. Se o usuário inserir dados incorretos, um pop-up de alerta será exibido. A tela de login é responsiva, utilizando media queries para adaptação a diferentes tamanhos de tela. Durante o desenvolvimento, foi possível aprofundar o conhecimento em HTML e CSS, especialmente na criação de formulários interativos e na implementação de soluções responsivas.
